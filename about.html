How We Got Here

Detection Methods. The detection methods usually use different features to train a binary classification model to distinguish real and fake faces. In this work, we implement representative detection methods that can be roughly divided into the following three categories. The first category of detection methods uses the signal-level artifacts (e.g., blending, warping, and shading artifacts) on a single frame that are introduced during the deepfake generation.We found that the deepfake algorithm can only generate images with limited resolution, which would bring the face warping artifacts. They directly used simple image processing to simulate such artifacts to generate negative samples and then trained a Convolutional Neural Network (CNN) model to capture the artifacts. Based on this, we are able to solve the multi-scale problems at the image and feature level.

Types of Identities

(1) Same identities: We directly select a frame from the driving video as the source image for deepfake generation. Therefore, the identities of the source images and the driving videos are the same in this setting. (2) Similar identities: After removing the data with the same identity in the candidate source image dataset, we select the source image that is most similar to the driving video but has different identities for deepfake generation.
(3) Dissimilar identities: In this setting, we select the source image that is least similar to the driving video from the candidate source image dataset for deepfake generation.
